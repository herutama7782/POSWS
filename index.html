Cara Cetak ke Printer Bluetooth

Agar aplikasi web ini bisa mencetak, perlu sedikit pengaturan satu kali di ponsel Anda.

  1. Pasangkan Printer: Buka Pengaturan Bluetooth di ponsel Anda, nyalakan printer, lalu 'Pair' atau 'Pasangkan' perangkat.
  2. Install Aplikasi 'Jembatan': Browser tidak bisa langsung terhubung ke printer. Anda perlu aplikasi 'Layanan Cetak' (Print Service) sebagai jembatan. Kami merekomendasikan:
  3. Install RawBT dari Play Store
  4. Siap Mencetak: Setelah terinstall, printer Bluetooth Anda akan muncul sebagai pilihan saat Anda menekan tombol "Cetak".
